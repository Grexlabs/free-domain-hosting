## 🌐 Free HTML Hosting (No Expiry)

Host plain HTML/CSS/JS files for free — great for landing pages, portfolios, or serving Claude-generated code directly to the web. No server-side scripting needed.

### 🖊️ How to Host a Plain HTML Page (Step-by-Step)

#### Option 1 – GitHub Pages (Recommended, Never Expires)
1. Create a repo named `yourusername.github.io`
2. Push your `index.html` to the `main` branch
3. Go to **Settings → Pages → Source → main branch**
4. Your site is live at `https://yourusername.github.io`

> ✅ No header/footer injected. Full control. Custom domain supported. Free SSL.

#### Option 2 – Blogger (Static Page Trick)
1. Create a Blogger blog at [blogger.com](https://blogger.com)
2. Go to **Pages → New Page**
3. Switch to **HTML mode** and paste your full HTML code
4. In **Page Settings**, disable header, footer, and sidebars (use a blank layout theme)
5. Publish — your page lives at `https://yourname.blogspot.com/p/page-name.html`

> ⚠️ Blogger injects some Google scripts. Use a custom theme to strip header/footer. Best for simple content pages.

#### Option 3 – Tiiny.host (Easiest, No Account Needed)
1. Go to [tiiny.host](https://tiiny.host)
2. Upload your `.html` file or `.zip`
3. Get a `yourname.tiiny.site` URL instantly

> ✅ Zero signup needed. No ads. Clean URL. Files stay up indefinitely on free tier.

#### Option 4 – Netlify Drop (Drag & Drop)
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag your HTML folder onto the page
3. Get a live URL instantly

> ✅ No login required for instant deploy. Create an account to keep the URL permanently.

#### Option 5 – Surge.sh (CLI)
```bash
npm install -g surge
surge /path/to/your/folder yourname.surge.sh
```
> ✅ Free, no expiry, custom domain supported with CNAME.
